SuDoKu

The Application

This application was made in Qt using c++ to create a fully functional Sudoku game and solver. The application can generate puzzles, import puzzles from a tab-delimited input file, and have a user hand enter a puzzle. The user can try to solve the puzzle, check if their solution is valid, or autosolve the puzzle

Compiling

The recommended way to compile would be to open the project file in QtCreator(it should have the relevant Qt Libraries) and build and run inside the creator.
